## Hướng dẫn chạy bài

### 1. Lab01
- Các file được viết độc lập theo từng bài.
- Có thể biên dịch/chạy từng file bằng lệnh:

```bash
javac bai01.java
java bai01
```

### 2. Lab02 (Flappy Bird)
- Di chuyển vào thư mục `Lab02`.
- Biên dịch và chạy chương trình:

```bash
javac *.java
java Main
```
